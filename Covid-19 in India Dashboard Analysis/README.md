# Covid-19 in India Dashboard Analysis

## Project Overview
This Tableau project presents a comprehensive analysis of Covid-19 cases in India using interactive visualizations. The dashboard helps in understanding the spread of Covid-19 across different states, testing details, vaccination progress, age group distribution, gender analysis, and state-wise death statistics.

## Key Features
- State-wise Covid confirmed, cured, and death cases
- Map visualization of total deaths by state
- Age group distribution of Covid cases
- Gender-wise analysis without missing values
- ICMR testing labs distribution across states
- State-wise testing details (Positive, Negative, Total Samples)
- First dose vs Second dose vaccination analysis
- Vaccine-wise doses administered (CoviShield, Covaxin, Sputnik V)

## Tools Used
- Tableau
- Excel / CSV Dataset
- Data Cleaning & Visualization Techniques

## Dashboard Insights
- Maharashtra has the highest number of ICMR testing labs
- CoviShield has the highest number of vaccine doses administered
- Male cases are higher compared to female cases
- Uttar Pradesh shows high testing sample counts
- Covid confirmed and cured cases increased rapidly during 2021

## Project Purpose
The main goal of this project is to provide a clear and interactive visual representation of Covid-19 trends in India, helping users quickly understand key insights and compare state-wise performance.

## Conclusion
This dashboard improves decision-making by transforming raw Covid-19 data into meaningful insights through interactive Tableau visualizations.
